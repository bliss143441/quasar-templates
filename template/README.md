# Quasar App

> A Quasar project

## Build Setup

``` bash
# install dependencies
$ npm install

# serve with ssr and hot reload at localhost:3030
$ npm run serve

# build for production with minification
$ quasar build

# lint code
$ quasar lint
```

## Run API Server with SSR

### For Development
In one terminal run

``` bash
$ npm run serve
```

### For Production

``` bash
$ quasar build
$ npm run prod
```
